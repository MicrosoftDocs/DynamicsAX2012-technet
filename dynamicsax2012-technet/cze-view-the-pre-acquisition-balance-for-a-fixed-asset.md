# (CZE) View the pre-acquisition balance for a fixed asset 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets** \> **Common** \> **Fixed assets** \> **Fixed assets**.

2.  Select a fixed asset group and click **Value models**.

3.  Select the value model and click **Inquiry** \> **Balances** to open the **Fixed asset balances** form.

4.  View the pre-acquisition balance for the fixed asset in the **Pre-Acquisition** field.

5.  Press CTRL+S or close the form.
