# Financials I license code (Ledger) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Financials I** license code enables access to some of the following General ledger functionality:

  - Bank information and documents

  - Bills of exchange

  - Budgeting

  - Checks

  - Collection letters

  - Credit cards

  - Currencies and reporting currencies

  - Free text invoice corrections

  - General ledger

  - Giro

  - Invoice journals

  - Invoice pools excluding posting

  - Invoice register and invoice approval journals

  - Journalizing

  - Negotiable instruments

  - Payment management

  - Payment schedules

  - Postdated checks

  - Prenotes

  - Promissory notes

  - Reversing entries

  - Sales taxes, sales tax jurisdictions, and withholding taxes

  - Transaction reversals
