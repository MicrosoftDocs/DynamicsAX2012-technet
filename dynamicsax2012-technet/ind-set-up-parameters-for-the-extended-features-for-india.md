# (IND) Set up parameters for the extended features for India 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



Use the **General ledger parameters** form to activate parameters for miscellaneous charges, financial statements, and fixed assets for India.

1.  Click **General ledger** \> **Setup** \> **General ledger parameters**.

2.  Under **Extended features for India** group field, select the **Miscellaneous charges** check box to activate the additional features required for miscellaneous charges.

3.  Select the **Financial statements** check box to activate the addition features in the financial statements for India.

4.  Click the **Fixed assets** tab.

5.  Select the **Companies Act Depreciation** check box to activate the India depreciation for Companies act feature and select the **Income tax Act depreciation** check box to activate the India depreciation for Income tax features and the Depreciation schedule report.

6.  Press CTRL+S or close the form.
