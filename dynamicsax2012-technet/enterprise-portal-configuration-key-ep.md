# Enterprise Portal configuration key (EP) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Enterprise Portal** configuration key controls access to Enterprise Portal for Microsoft Dynamics AX.

You cannot enable or disable this key in the **License configuration** form.

The following is a subset of the forms available when the configuration key is enabled.
