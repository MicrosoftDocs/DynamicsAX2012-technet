# Registration employees license code (SFCUsers) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Registration employees** license code enables access to forms and functions for enabling workers to register time and attendance information.

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:

[Registration employees configuration key (JmgRegistrationEmployees)](registration-employees-configuration-key-jmgregistrationemployees.md)
