# Credit card configuration key (CreditCard) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Credit card** configuration key controls access to credit card authorization forms and functions. For more information, see [About credit card authorizations](about-credit-card-authorizations.md). If the **Credit card** configuration key is enabled, the following fields, tabs, and buttons are available:

  - The **Other** FastTab is displayed in the **Terms of payment** form.

  - The **Credit card number** field is displayed in the **Sales order** form.

  - The **Process shipment** button is displayed in the **Shipment** form.

You can enable or disable this key in the **License configuration** form.

The following forms are available when the configuration key is enabled.
