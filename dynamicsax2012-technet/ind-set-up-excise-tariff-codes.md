# (IND) Set up excise tariff codes [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



Use the **Excise tariff codes** form to define excise tariff codes. The excisable items are included under the excise tariff code.

1.  Click **General ledger** \> **Setup** \> **Sales tax** \> **India** \> **Excise tariff codes**.

2.  Enter the excise tariff code in the **Tariff code** field and the description of the excise tariff code in the **Description** field.
