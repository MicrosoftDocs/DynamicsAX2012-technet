# (CHN) Approve a journal 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **General ledger** \> **Journals** \> **General journal**.
    

    > [!NOTE]
    > <P>For more information, see <A href="https://technet.microsoft.com/en-us/library/aa557917(v=ax.60)">Journal header (form)</A>.</P>



2.  In the **Show** field, select **Open** to display a list of journal vouchers that have a status of Open.

3.  Select a journal voucher from the list and click **Validate** to validate the journal voucher.

4.  Click **Approval** and select **Report as ready**. The approval status is now set to **Ready**.
    

    > [!NOTE]
    > <P>You can reject an approved journal voucher by selecting <STRONG>Approval</STRONG> and then <STRONG>Reject</STRONG>. When a journal voucher with an approval status of <STRONG>Ready</STRONG> or <STRONG>Rejected</STRONG> is modified, the approval status is changed to <STRONG>Open</STRONG>.</P>



5.  Click **Approval** and select **Approve** to open the **Journal approval** form.
    

    > [!NOTE]
    > <P>For more information, see <A href="approve-a-journal.md">Approve a journal</A>.</P>
