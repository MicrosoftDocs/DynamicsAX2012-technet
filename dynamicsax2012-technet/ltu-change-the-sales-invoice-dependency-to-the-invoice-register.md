# (LTU) Change the sales invoice dependency to the invoice register 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

If a posted invoice must not be included in the invoice register, change the dependence to the register for that invoice.


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Accounts receivable** \> **Inquiries** \> **Journals** \> **Invoice journal**.

2.  Select the invoice for which to change the dependence, and click **Function** \> **Change dependency to register**.

3.  Click **Yes** to **Change dependency to registry**. The **Include in register** check box is automatically selected to include the document in the register log.

4.  Press CTRL+S or close the form.
