# Change a formula line in a batch order 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to make changes to a formula line in a batch order.

1.  Click **Production control** \> **Common** \> **Production orders** \> **All production orders**.

2.  Double-click the batch order to display details for the batch order on the **Batch order** form.

3.  On the **Action Pane**, on the **Production order** tab, click **Order details** and then click **Formula**. The **Formula lines-batch** form is displayed.

4.  Select the formula line that you want to change.

5.  Change the desired field values by using the applicable tabs in the form.
