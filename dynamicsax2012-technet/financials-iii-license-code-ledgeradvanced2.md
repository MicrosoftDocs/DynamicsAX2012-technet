# Financials III license code (LedgerAdvanced2) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Financials III** license code enables access to some of the following functionality in General ledger:

  - Budget control

  - Centralized customer statements

  - Collections

  - Encumbrance processes

  - Generated allocations

  - Pre-encumbrance processes
