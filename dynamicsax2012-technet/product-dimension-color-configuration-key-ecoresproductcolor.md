# Product dimension - color configuration key (EcoResProductColor) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Product dimension - color** configuration key controls access to the **Color** product dimension.


> [!NOTE]
> <P>The <STRONG>Color</STRONG> product dimension can be used for other purposes than color variants, and it can be renamed.</P>



You can enable or disable this key in the **License configuration** form.
