# Czech Republic 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

This section of Help provides information about functionality and features in Microsoft Dynamics AX that are specific to the Czech Republic. Procedures or other Help topics for forms and reports that are specific to the Czech Republic include (CZE) in the topic title. Functionality for the Czech Republic is available only to legal entities whose primary address is in the Czech Republic.

## Localizations

For a list of planned and released features that are specific to the Czech Republic, refer to the [Czech Republic localization report](https://mbs.microsoft.com/files/customer/ax/support/supportnews/czechrepublic.html).

For additional information about localization features, refer to the [Microsoft Dynamics Localization Portal](https://mbs.microsoft.com/customersource/northamerica/ax/support/support-news/gfmlocalizationportalmc).

## Modified forms

Forms can be modified to provide functionality for specific business situations or for specific countries or regions. If a modified form exists, you can view Help for the standard form or the modified form when you press F1.
