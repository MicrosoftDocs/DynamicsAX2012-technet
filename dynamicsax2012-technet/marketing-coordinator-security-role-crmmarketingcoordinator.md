# Marketing coordinator security role (CRMMarketingCoordinator) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Marketing coordinator security role represents a user who produces and distributes marketing materials.
