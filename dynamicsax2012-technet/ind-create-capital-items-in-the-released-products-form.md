# (IND) Create capital items in the Released products form 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Product information management** \> **Common** \> **Released products**.

2.  Click **Product**, to open the **Create product** form.

3.  Select the inventory model group that is created for capital items in the **Item model group** field.

4.  Click the **Purchase** FastTab.

5.  Create a capital item and select the item group that is created for capital items in the **Item group** field.
    

    > [!NOTE]
    > <P>The <STRONG>Asset group</STRONG> check box must be selected for the item group in the <STRONG>Item groups</STRONG> form, to be identified as a capital item group.</P>
