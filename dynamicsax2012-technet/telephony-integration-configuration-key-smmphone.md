# Telephony integration configuration key (smmPhone) 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Telephony integration** configuration key allows you to make telephone calls from Microsoft Dynamics AX if the required hardware and software for Telephony Application Programming Interface (TAPI) are installed.

You can enable or disable this key in the **License configuration** form.
