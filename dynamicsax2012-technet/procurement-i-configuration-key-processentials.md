# Procurement I configuration key (ProcEssentials) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Procurement I** configuration key controls access to base features in the Procurement and sourcing module in Microsoft Dynamics AX and Enterprise Portal for Microsoft Dynamics AX.
