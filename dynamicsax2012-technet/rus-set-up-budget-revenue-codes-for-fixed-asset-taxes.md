# (RUS) Set up budget revenue codes for fixed asset taxes 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Cash and bank management** \> **Setup** \> **Payment order** \> **Budget revenue classification**.

2.  Press CTRL+N to create a new line.

3.  In the **Budget revenue code** field, enter the budget revenue code for the fixed asset tax.

4.  In the **Name** field, enter the description for the budget revenue code.

5.  Select the **SSGS** check box to specify if the budget revenue code belongs to the Standard System of Gas Supply (SSGS).

6.  Press CTRL+S or close the form.
