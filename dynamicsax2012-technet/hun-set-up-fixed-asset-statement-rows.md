# (HUN) Set up fixed asset statement rows [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets** \> **Setup** \> **Fixed asset statement rows**.
    

    > [!NOTE]
    > <P>For more information, see "Fixed asset statement rows (form)" in the Applications and Business Processes Help.</P>



2.  In the **Row name** field, enter the name of the fixed asset statement row.

3.  In the **Row type** field, select the row type as **Total**.

4.  Click **Totals**, and then select the fixed asset accounts interval in the **From** and **To** fields in the **Fixed asset interval** form.

5.  Click **Up** to move one row upward.

6.  Click **Down** to move one row downward.

7.  Press CTRL+S or close the form.
