# Sales representative security role (TradeSalesRepresentative) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Sales representative security role represents a user who documents sales events and responds to sales inquiries.
