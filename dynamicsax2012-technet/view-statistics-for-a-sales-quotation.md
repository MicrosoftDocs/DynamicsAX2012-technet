# View statistics for a sales quotation [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Sales and marketing** \> **Periodic** \> **Sales management** \> **Management statistics**.
    

    > [!IMPORTANT]
    > <P>You must create default data for a query before you can view statistics. For more information, see <A href="queries.md">Queries</A>.</P>



2.  In the **Management statistics** form, press CTRL+N to create a new version, and give it a meaningful name and description.

3.  On the **General** tab, enter your criteria. For more information about the criteria, see [Management statistics (form)](https://technet.microsoft.com/en-us/library/aa582715\(v=ax.60\)).

4.  Click **Calculate**, and then select **All** to calculate data for all queries, or enter a **Query name** to select a specific query.

5.  Click **Graph**, and then select **All** to show statistics for all queries in a graph, or enter a **Query name** to select a specific query.
