# Over/Under Delivery configuration key (LogisticsOverUnderDelivery) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Over/under delivery** configuration key controls access to the forms and functions that are used to accept overdeliveries and underdeliveries relative to the ordered quantity.


> [!NOTE]
> <P>If this configuration key is disabled, no overdeliveries and underdeliveries are allowed. Disabling the configuration key corresponds to setting the overdelivery and underdelivery percentage to 0. Partial deliveries are not controlled by this configuration key.</P>



You can enable or disable this key in the **License configuration** form.
