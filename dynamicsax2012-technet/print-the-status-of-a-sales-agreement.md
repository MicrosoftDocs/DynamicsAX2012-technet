# Print the status of a sales agreement 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Sales and marketing** \> **Reports** \> **Sales agreements** \> **Sales agreement fulfillment**.

2.  Print the report by using one of the following methods:
    
      - Click **OK** to print a report that displays the status of all sales agreements.
    
      - Click **Select** to define criteria for the report. For example, you can print the status of a specific sales agreement. When you have finished defining criteria, click **OK** to print the report.
