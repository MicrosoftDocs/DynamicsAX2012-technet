# Connector administrator security role (ConnectorAdministrator) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3_

The Connector administrator security role represents a user who maintains access to connector APIs.


> [!NOTE]
> <P>This role is available only if Microsoft Dynamics AX 2012 R3 is installed.</P>
