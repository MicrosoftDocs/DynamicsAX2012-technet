# Create price groups 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

You can create price groups for those groups of customers that you want to handle in a similar way regarding price.

1.  Click **Inventory management** \> **Setup** \> **Price/discount** \> **Customer price/discount groups**.

2.  In the **Show** filter, select **Price group**.

3.  Click **New** to create a new line.

4.  Enter a price group code and a name for the price group.
