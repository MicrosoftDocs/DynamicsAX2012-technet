# Use a wizard to create item coverage settings [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use a wizard to create item coverage settings.

1.  Click **Product information management** \> **Common** \> **Released products**.

2.  From the **Released products** list page, select the item to create coverage dimensions for.

3.  On the **Action Pane**, click the **Plan** tab. In the **Coverage** group, click the **Item coverage** button.

4.  In the **Item coverage** form, click the **Wizard** button to create coverage parameters for the item.

5.  After you have finished running the wizard, you can modify the item coverage parameters in the **Item coverage** form.
