# (RUS) Set up fixed asset module parameters 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets (Russia)** \> **Setup** \> **Parameters**.

2.  Click the **Tax accounts** tab and, in the **Sales tax code** field, select a sales tax code for assessed tax, transport tax, or land tax.

3.  In the **Compression** field, select the level of compression for the fixed asset transactions.

4.  Click the **Number sequences** tab and, in the **Number sequence code** field, select the number sequence code for **Assessed tax register journal number**.

5.  Press CTRL+S or close the form.
