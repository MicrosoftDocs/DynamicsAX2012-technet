# Sales force automation license code (smmSalesForceAutomation) 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Sales force automation** license code enables access to the sales features of Microsoft Dynamics AX.

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:

[Sales and marketing series configuration key (SmmCRM)](sales-and-marketing-series-configuration-key-smmcrm.md)

[Sales Representative configuration key (EPSalesRep)](sales-representative-configuration-key-epsalesrep.md)


> [!NOTE]
> <P>The configuration keys listed here may have additional child configuration keys. Click the link for more information about a configuration key.</P>
