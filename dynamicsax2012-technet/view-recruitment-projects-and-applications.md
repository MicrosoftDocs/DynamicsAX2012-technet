# View recruitment projects and applications 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use the **Recruitment projects** page to view a list of recruitment projects for which you are the hiring manager or the alternative contact person.


> [!NOTE]
> <P>Depending on your role or the privileges that are assigned to you, you might need to go to your Employee services site before you complete the procedure in this topic.</P>



## View applications for a recruitment project

1.  Click **Team** on the top link bar, and then click **My recruitment projects** on the Quick Launch.

2.  Select the recruitment project that you want to view applications for.

3.  On the Action Pane, click **Applications**. The **Applications** page is displayed, where you can view a list of applications that were submitted for the selected recruitment project.
