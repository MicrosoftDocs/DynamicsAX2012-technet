# Trade agreements configuration key (TradeAgreements) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Trade agreements** configuration key controls access to trade agreements such as prices and discounts. It also controls access to supplementary items in the **Procurement and sourcing**, **Inventory management**, **Sales and marketing**, **Accounts receivable**, **Trade allowance management**, and **Call center** modules.

You cannot enable or disable this key in the **License configuration** form.

The following forms are available when the configuration key is enabled.
