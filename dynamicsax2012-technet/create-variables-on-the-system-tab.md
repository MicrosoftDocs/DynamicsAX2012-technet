# Create variables on the System tab 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_


> [!NOTE]
> <P>This information applies only to Product builder.</P>



1.  Click **Product information management** \> **Common** \> **Product builder** \> **Product models**.

2.  Select a **Product model**.

3.  On the **Action Pane**, click the **Models** tab. In the **Modify** group, click **Open**.

4.  Select the **System** tab and press CTRL+N to create a new line.

5.  Select the variable type in the **System variable** field.

6.  In the **Variable** field, enter the name of the system variable.
