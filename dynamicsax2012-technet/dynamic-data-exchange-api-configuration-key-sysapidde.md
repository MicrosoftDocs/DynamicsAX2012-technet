# Dynamic Data Exchange API configuration key (SysAPIDDE) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Dynamic data exchange api** configuration key controls access to the Dynamic Data Exchange (DDE) classes.

The **Dynamic data exchange api** configuration key is disabled by default for security reasons.

You can enable or disable this key in the **License configuration** form.
