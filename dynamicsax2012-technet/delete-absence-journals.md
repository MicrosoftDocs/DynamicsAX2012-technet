# Delete absence journals 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

You can delete an absence journal only if you have not yet transferred the absence journal for approval.


> [!NOTE]
> <P>If an absence journal contains approved future absences, you must cancel the approval of the future absences before you can delete the absence journal. For more information, see <A href="approve-or-reject-absences.md">Approve or reject absences</A>.</P>



### Delete an absence journal

1.  Click **Human resources** \> **Periodic** \> **Absence** \> **Delete absence journals**.

2.  Choose one of the following actions:
    
      - Click **OK** to delete all absence journals.
    
      - Click **Select** to select the employees to delete absence journals for.
