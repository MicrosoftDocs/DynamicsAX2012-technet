# Submit a document (Enterprise Portal) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

To submit a document to the workflow system for processing, click the **Submit** button on the workflow message bar of the page that you are on.

For example, to submit a purchase requisition, click **Submit** on the **Purchase requisitions** page.
