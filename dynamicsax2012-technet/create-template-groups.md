# Create template groups 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

To use quotation templates, you must organize them into template groups. You must create the template groups before you can create quotation templates.

1.  Click **Sales and marketing** \> **Setup** \> **Quotations** \> **Template groups**.

2.  Press CTRL+N to create a new template group.

3.  In the **Group ID** field, enter a unique name for the template group.

4.  In the **Description** field, enter a description of the template group.

5.  Close the form to save your changes.
