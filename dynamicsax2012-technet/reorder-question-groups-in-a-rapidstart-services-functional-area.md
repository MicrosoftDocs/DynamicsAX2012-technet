# Reorder question groups in a RapidStart Services functional area 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Beginning October 1, 2015, RapidStart services will no longer be supported. For more information, see the announcement on CustomerSourcehttps://mbs.microsoft.com/partnersource/northamerica/news-events/news/RapidStartServicesDiscontinued or PartnerSourcehttps://mbs.microsoft.com/customersource/northamerica/news-events/news-events/news/RapidStartServicesDiscontinued.

You can use the Design area in Microsoft Dynamics ERP RapidStart Services to change the order of question groups as they are listed in the functional area. The changes that you make are reflected in the Configure area in RapidStart Services.


> [!NOTE]
> <P>You cannot reorder question groups that belong to a published functional area. For information about how to unpublish a functional area, see <A href="unpublish-a-functional-area-in-rapidstart-services.md">Unpublish a functional area in RapidStart Services</A>.</P>



1.  In the RapidStart Services header, click **Design**, and then click **Functional area**.

2.  In the **Functional area** list, select the functional area for which you want to reorder question groups.

3.  Click the **Functional area** button, and then click **Reorder question groups**.

4.  In the **Reorder question groups** form, select the name of a question group, and then click the **Up** or **Down** button to move it to a different position in the list.

5.  Repeat step 4 for each question group that you want to move.

6.  Click **Reorder**.
