# Environmental clerk security role (EMSEnvironmentalClerk) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Environmental clerk security role represents a user who documents environmental sustainability events.
