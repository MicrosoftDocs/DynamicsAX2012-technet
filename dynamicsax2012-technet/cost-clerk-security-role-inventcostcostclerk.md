# Cost clerk security role (InventCostCostClerk) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Cost clerk security role represents a user who authorizes and maintains costs, inventory valuations, and cost accounting calculations; responds to costs, inventory valuations, and cost accounting inquiries.
