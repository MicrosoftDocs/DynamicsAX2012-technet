# (CHN) Generate a group of voucher types 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



Use the **Voucher type setup wizard** to automatically generate a group of voucher types for a company. The wizard cannot generate the same voucher type twice for the company. However, you can modify the existing voucher type in the **Voucher type setup** form.


> [!NOTE]
> <P>You must set up the chart of accounts before generating the wizard.</P>



1.  Click **General ledger** \> **Setup** \> **Journals** \> **Voucher type setup** \> **Voucher type setup wizard**.

2.  Select the group of vouchers to be created and click **Next**.

3.  Click **Finish** to save and close the wizard.
