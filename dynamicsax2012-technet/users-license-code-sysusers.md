# Users license code (SysUsers) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Users** license code enables access to some administrative features of Microsoft Dynamics AX.

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:

[Administration configuration key (SysAdmin)](administration-configuration-key-sysadmin.md)

[Country/Regional specific features configuration key (CRSECountry)](country-regional-specific-features-configuration-key-crsecountry.md)

[Record level security configuration key (SysRecordLevelsecurity)](record-level-security-configuration-key-sysrecordlevelsecurity.md)

[Reporting Services configuration key (ReportingServices)](reporting-services-configuration-key-reportingservices.md)

[Optional APIs configuration key (SysAPI)](optional-apis-configuration-key-sysapi.md)


> [!NOTE]
> <P>The configuration keys listed here may have additional child configuration keys. Click the link for more information about a configuration key.</P>
