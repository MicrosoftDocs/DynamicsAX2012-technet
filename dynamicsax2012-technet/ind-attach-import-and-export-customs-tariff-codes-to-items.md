# (IND) Attach import and export customs tariff codes to items 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Product information management** \> **Common** \> **Released products**. Create a new item or double-click an existing item.

2.  Click the **Foreign trade** FastTab, and then select the import tariff code for the item in the **Import tariff code** field.
    

    > [!NOTE]
    > <P>You cannot attach customs tariff codes to items of type <STRONG>Service</STRONG>.</P>



3.  Select the export tariff code for the item in the **Export tariff code** field.
