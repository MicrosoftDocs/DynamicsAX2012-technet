# (RUS) Hide the register fields 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

Before generating the registers, you can elect to hide fields that you might not need. You can also change the display options for fields that are set up when you create a register.


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **General ledger** \> **Journals** \> **Tax register journal**. Create a journal line, and then click **Lines**.

2.  Select the register for which you want to hide the register fields, and then click the **Hide** tab.

3.  Press CTRL+N to create a new line.

4.  In the **Register field** field, select the fields to be excluded from the registry.
    

    > [!NOTE]
    > <P>Only the fields that are set up for the corresponding register are available for selection.</P>



5.  Select the **Hide** check box to hide the field in the register.

6.  Press CTRL+S or close the form.
