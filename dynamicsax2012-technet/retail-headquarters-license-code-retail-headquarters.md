# Retail headquarters license code (Retail Headquarters) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack_

The **Retail Headquarters** license code controls access to all of the retail features in Microsoft Dynamics AX and Enterprise Portal for Microsoft Dynamics AX.

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:
