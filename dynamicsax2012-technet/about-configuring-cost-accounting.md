# About configuring cost accounting 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Microsoft Dynamics AX offers many configuration and setup options for cost accounting, but only some of them are necessary for the mandatory or basic setup.


> [!NOTE]
> <P>In general, you do not have to follow the order in which these steps are described. However, because of certain interdependencies, you may have to perform one step before another.</P>



To simplify the configuration process, the process has been divided among the following procedural topics:
