# Unsolicited vendor registration configuration key (VendUnsolicitedRegistration) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Purchase requisition** **Unsolicited vendor registration** configuration key controls access to the **Unsolicited vendor registration** page in Enterprise Portal for Microsoft Dynamics AX.

The **Unsolicited vendor registration** page allows vendors in the marketplace who are not vendors for your organization to register and provide details about their products and services. Employees, when searching for a vendor from whom they can buy goods or services, can search for current vendors or unsolicited vendors. If the vendor is an unsolicited vendor, the employee can create a new vendor request, which is vetted by purchasing agents to determine whether the unsolicited vendor meets the buying needs of the employee and the larger organization.

You can enable or disable this key in the **License configuration** form.
