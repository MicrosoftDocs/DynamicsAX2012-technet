# What's new: Country-specific features for Malaysia (MYS) [AX 2012]


In Microsoft Dynamics AX 2012 R3, we changed and added country-specific functionality for Malaysia. For more information, see the table that follows.

## What’s new in cumulative update 8 for Microsoft Dynamics AX 2012 R3

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>What’s new</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>Print sales invoices and purchase invoices that include details about goods and services tax (GST).</p></td>
<td><p>You can generate and print self-billed GST invoices for goods and services that you purchase. You can also include a GST relief clause in a customer invoice and specify the printing format of invoices. GST invoices are printed according to Malaysian GST regulations.</p></td>
</tr>
<tr class="even">
<td><p>GST details for customer invoices, free text invoices, or vendor invoices</p></td>
<td><p>You can generate and print GST details in a credit note or debit note that is created for a customer invoice, free text invoice, vendor invoice, or project invoice.</p></td>
</tr>
</tbody>
</table>
