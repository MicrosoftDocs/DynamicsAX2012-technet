# Process manufacturing production license code (ProcessIndustriesManufacturing) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Process Manufacturing Production** license code enables access to the production-related features of process manufacturing in Microsoft Dynamics AX.
