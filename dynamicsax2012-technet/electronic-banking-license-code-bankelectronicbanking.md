# Electronic banking license code (BankElectronicBanking) 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Electronic banking** license code enables access to the electronic banking features in Microsoft Dynamics AX.

This type of license is required for users who create remittance files for customers and vendors.
