# (RUS) Create a register using the wizard 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



Use the register creation wizard to create registers manually. The value of these registers is stored on a periodic basis in a tax register journal, and their data can be used to prepare tax statement reports or for setting up calculation sequences.

1.  Click **General ledger** \> **Setup** \> **Profit tax** \> **Registers**. Click **Wizard** to open the Tax register creation wizard form

2.  Click **Next** to create a new register.

3.  In the **Register type** field, enter the type of the register.
    

    > [!NOTE]
    > <P>The name must contain numerical or alphabetical values (Latin letters only). The value that is entered will be added to the drop-down list of registers that is used to create registers in the <STRONG>Tax registers</STRONG> form.</P>



4.  In the **Register name** field, enter the name of the register.

5.  In the **Description** field, enter a description for the tax register.

6.  In the **Extended description** field, enter additional information for the tax register.

7.  Click **Next** to set up register fields.

8.  In the **Type** field, select the data type of the register fields. All available Microsoft Dynamics AX field types can be selected.

9.  In the **Name** field, modify the selected register field name, if needed.

10. Click **Next** to complete the wizard.

11. Click **Finish** to create and compile the new tax register.
