# Cost accounting IV license code (COSPlanCostCalc) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Cost accounting IV** license code enables access to some of the following Cost accounting functionality:

  - Budget

  - Calculation setup

  - Cost allocation

  - Dimension setup

  - Flexible budgeting

  - Workload

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:
