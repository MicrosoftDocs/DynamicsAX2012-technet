# Process engineering manager security role (RouteProcessEngineeringManager) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Process engineering manager security role represents a user who reviews new products, materials, and processes.
