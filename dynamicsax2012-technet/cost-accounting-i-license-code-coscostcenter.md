# Cost accounting I license code (COSCostCenter) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Cost accounting I** license code enables access to some of the following Cost accounting functionality:

  - Accrual schemes

  - Accruals

  - Allocations

  - Budgets

  - Calculation transactions

  - Cost accounting parameters

  - Cost balances

  - Cost categories

  - Cost lines

  - Cost rates

  - Cost splitting

  - Cost transactions

  - Cost dimensions

  - Cost display

  - Expense distribution sheets

  - Hierarchies

  - Journals

  - Lines

  - Period balances

  - References

  - Reports

  - Service budgets

  - Service transactions

  - Special calculations

  - Validation lists
