# Create a commodity pricing template [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to create a commodity pricing template.

1.  Click **Inventory management** \> **Setup** \> **Commodity pricing** \> **Pricing template**.

2.  Press CTRL+N to add a new pricing template line.

3.  In the **Pricing template** field, enter an identifier or name for the commodity pricing template.

4.  In the **Description** field, enter a description for the pricing template.
