# (IND) About customer payments 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

This topic provides specific information about customer payments for legal entities whose primary address is in India. For full documentation about customer payments, see the following topics:

[Receive and enter customer payments](receive-and-enter-customer-payments.md)

[Key tasks: Customer payments and settlements](key-tasks-customer-payments-and-settlements.md)

## Mark invoice lines for settlement

When you process a customer payment, you can use the **Mark invoice lines** form to review the TDS and TCS amounts for individual transaction lines, and then select the lines for settlement. The amount to settle value for an invoice line is based on the following calculations:

**Amount to settle** = **Line amount** - **TDS**

–or–

**Amount to settle** = **Line amount** + **TCS**

For more information, see [Mark invoice lines (form)](https://technet.microsoft.com/en-us/library/hh227459\(v=ax.60\)).
