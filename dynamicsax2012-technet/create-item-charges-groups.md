# Create item charges groups 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Charges groups help you group the various types of charges for items.

1.  Click **Accounts payable** \> **Setup** \> **Charges** \> **Item charge groups**.
    
    –or–
    
    Click **Accounts receivable** \> **Setup** \> **Charges** \> **Item charge groups**.

2.  Click **New** to create an item charges group.

3.  In the **Charges group** field, enter a code for the group. The code can be alphanumeric.

4.  In the **Description** field, enter a description for the group.
