# (IND) Enable customs and EXIM incentive schemes 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

Before you can use customs and export/import (EXIM) incentive schemes, you must enable the incentive schemes in the **General ledger parameters** form.

1.  Click **General ledger** \> **Setup** \> **General ledger parameters**.

2.  In the **General ledger parameters** form, click **Sales tax**.

3.  On the **General** FastTab, in the **Apply India taxes** section, select the **Customs** check box.

4.  In the **EXIM** section, select the **Activate incentive schemes** check box.

The **Incentive schemes** button is now available on the **General** FastTab.
