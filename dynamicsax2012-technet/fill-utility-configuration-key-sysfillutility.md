# Fill utility configuration key (SysFillUtility) 


_**Applies To:** Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_


> [!WARNING]
> <P>This configuration key is not available if Microsoft Dynamics AX 2012 R2 is installed.</P>



The **Fill utility** configuration key enables the Fill utility functionality in Microsoft Dynamics AX.

You can enable or disable this key in the **License configuration** form.
