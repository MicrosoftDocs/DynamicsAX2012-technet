# (RUS) Set up the name for fixed asset tax transactions journals 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **General ledger** \> **Setup** \> **Journals** \> **Journal names**.

2.  Press CTRL+N to create a new name for fixed asset tax transactions.
    

    > [!NOTE]
    > <P>For more information, see "Set up journal names" in the Applications and Business Processes Help.</P>



3.  In the **Name** field, enter a journal name.

4.  In the **Journal type** field, select the type of fixed asset tax from the following options:
    
      - **Assessed tax**
    
      - **Transport tax**
    
      - **Land tax**

5.  In the **Voucher series** field, select the number sequence code for the journal.

6.  Press CTRL+S or close the form.


> [!NOTE]
> <P>Each journal template contains the voucher number sequence, the corresponding default account, and other details that are reflected in all journals that are created when the journal template is used.</P>
