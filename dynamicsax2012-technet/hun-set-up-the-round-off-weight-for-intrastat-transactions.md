# (HUN) Set up the round-off weight for Intrastat transactions 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Organization administration** \> **Setup** \> **Foreign trade** \> **Foreign trade parameters**.

2.  Click the **Round-off** FastTab.

3.  In the **Weight rounding** field, enter the rounding quantity.

4.  Select the **Round weight to 1** check box to round the weight of the item to 1 unit of weight.

5.  In the **Weight rounding form** field, select the rounding method as **Normal**, **Downward**, or **Rounding-up.**
    

    > [!NOTE]
    > <P>If the net weight is entered in fractions, the net weight is rounded to three decimal points.</P>



6.  Press CTRL+S or close the form.
