# Run forecast scheduling 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

When you have set up your demand, you can run forecast scheduling to calculate gross requirements for materials and capacity and to generate planned orders.

1.  Click **Master planning** \> **Periodic** \> **Forecast scheduling**.

2.  Select a forecast plan and an item.

3.  Click **OK**.

4.  To view the requirements that are calculated, use the **Released products** form. Click **Product information management** \> **Common** \> **Released products**. From the **Released products** list page, select a product and click the **Plan** tab. In the **Requirements** group, click the **Gross requirement** button.

5.  To view the planned orders that are generated, use the **Planned orders** form. Click **Master planning** \> **Common** \> **Planned orders**. Select the appropriate forecast plan.
