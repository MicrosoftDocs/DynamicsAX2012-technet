# Course confirmation list (HcmCourseConfirmation) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this report to print a letter you can send to people to confirm their registration for a particular training course. One confirmation letter is printed for each participant.
