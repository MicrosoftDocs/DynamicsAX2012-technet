# Product Builder/Dynamics AX 2009 Shared Tables configuration key (PBA\_SharedProductBuilder) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Product Builder / Dynamics AX 2009 shared tables** configuration key controls access to Product builder forms and functions.

You cannot enable or disable this key in the **License configuration** form.
