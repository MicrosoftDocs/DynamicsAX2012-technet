# Change the status of a recruitment project 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Before you can change the status of a recruitment project to **Finished** or **Canceled**, all applications related to the project must be closed.

Recruitment projects can have the following statuses:

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>Status</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p><strong>Schedule</strong></p></td>
<td><p>This status is displayed automatically when you create a project.</p></td>
</tr>
<tr class="even">
<td><p><strong>Start</strong></p></td>
<td><p>You have planned the project, created positions and media, and are ready to enter applications for the project.</p></td>
</tr>
<tr class="odd">
<td><p><strong>Finish</strong></p></td>
<td><p>All applications have been fully processed.</p></td>
</tr>
<tr class="even">
<td><p><strong>Cancel</strong></p></td>
<td><p>The project is no longer needed.</p></td>
</tr>
</tbody>
</table>


1.  Click **Human resources** \> **Common** \> **Recruitment** \> **Recruitment projects**.

2.  Select the recruitment project to update the status of.

3.  On the **Action Pane**, click **Recruitment project status**, and then select the new status of the project.
