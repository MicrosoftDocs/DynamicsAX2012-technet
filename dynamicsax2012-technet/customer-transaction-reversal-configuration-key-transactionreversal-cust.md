# Customer transaction reversal configuration key (TransactionReversal\_Cust) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Customer transaction reversal** configuration key controls access to transaction reversal forms and functions. When this configuration key is enabled, the **Reverse transaction** button is available in the **Customer transactions** form.

You can enable or disable this key in the **License configuration** form.

## Forms enabled or affected by the configuration key

The following forms are available or affected when the configuration key is enabled.
