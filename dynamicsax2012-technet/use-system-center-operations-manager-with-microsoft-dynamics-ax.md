# Use System Center Operations Manager with Microsoft Dynamics AX 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The System Center Management Pack for Microsoft Dynamics AX 2012 provides an end-to-end monitoring solution for Microsoft Dynamics AX. The management pack automatically discovers the entire Microsoft Dynamics AX environment, and monitors the configuration, availability, and performance of components. The management pack provides early warnings that an operator can use to proactively identify issues that can affect the performance and availability of the Microsoft Dynamics AX system.

For more information, see [System Center Management Pack for Microsoft Dynamics AX 2012](http://go.microsoft.com/fwlink/?linkid=393811%26clcid=0x409).
