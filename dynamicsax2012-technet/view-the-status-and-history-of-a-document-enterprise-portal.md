# View the status and history of a document (Enterprise Portal) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

You can use the **Workflow history details** page to view the history of a document that has been submitted to the workflow system for processing. This includes information about which user created the document, the time and date when it was submitted, a list of actions associated with the document, and any comments that were entered during the workflow process.

1.  Click **Home** on the top link bar, and then click **Workflow history** on the Quick Launch. The **Workflow history** page is displayed.

2.  Select a document in the list and then click **Actions** \> **View workflow details**. The **Workflow history details** page is displayed, where you can view more information about the document or recall the document.
