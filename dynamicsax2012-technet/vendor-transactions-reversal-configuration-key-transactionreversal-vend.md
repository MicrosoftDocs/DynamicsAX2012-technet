# Vendor transactions reversal configuration key (TransactionReversal\_Vend) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Vendor transaction reversal** configuration key controls access to transaction reversal forms and functions. When this configuration key is enabled, the **Reverse transaction** option is available in the **Vendor transactions** form.

You can enable or disable this key in the **License configuration** form.
