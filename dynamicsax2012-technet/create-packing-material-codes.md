# Create packing material codes 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Inventory management** \> **Setup** \> **Packing material** \> **Packing material codes**.

2.  Press CTRL+N to create a packing material code.

3.  In the **Code** field, type a code for the material.

4.  In the **Packing material** field, type a description of the material.

5.  In the **Unit** field, select a unit.

6.  Close the form to save your changes.
