# (RUS) Generate a profit tax declaration 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



Before you generate a profit tax declaration, you must set up the profit tax declaration report by using the Russian financial reports generator, and then create and calculate the tax register journal.

1.  Click **General ledger** \> **Reports** \> **External** \> **Financial reports generator**.

2.  Select the line that corresponds to the tax declaration.
    

    > [!NOTE]
    > <P>When you create a statement, the ledger periods are converted to tax periods that are used in the tax registers.</P>



3.  Click **Print** to open the **Base date** form.

4.  In the **Base date** field, select the date for report generation.

5.  Click **OK** to generate the report.
