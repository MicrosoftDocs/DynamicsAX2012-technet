# Product dimension - size configuration key (EcoResProductSize) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Product dimension - size** configuration key controls access to the **Size** product dimension.


> [!NOTE]
> <P>The <STRONG>Size</STRONG> product dimension can be used for other purposes than size variants, and it can be renamed.</P>



You can enable or disable this key in the **License configuration** form.
