# Delete a batch attribute group from an item 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to delete a batch attribute group from an item.

1.  Click **Product information management** \> **Common** \> **Released products**.

2.  Select the item with the batch attribute group that you want to delete.

3.  On the **Action Pane**, on the **Manage inventory** tab, in the **Batch attributes** group, click **Product specific**.

4.  Select the batch attribute group line that you want to delete.

5.  Click **Delete**. To delete the attribute group, click **Yes** to the **Delete record** prompt.
