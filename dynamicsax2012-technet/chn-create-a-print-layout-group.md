# (CHN) Create a print layout group 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **General ledger** \> **Setup** \> **Journals** \> **Voucher type setup** \> **Print layout**.

2.  On the **Overview** tab, press CTRL+N to create a new print layout.

3.  Enter the print layout group name and a short description for it in the **Print layout group** and **Description** fields.

4.  Click the **Setup** tab, select a **Print layout code** and enter a description for it.
    

    > [!NOTE]
    > <P>The TR layout series prints the report in the default currency and the TW layout series prints the report in both the foreign and default currencies.</P>
