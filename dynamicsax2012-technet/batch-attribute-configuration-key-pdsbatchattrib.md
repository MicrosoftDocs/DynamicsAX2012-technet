# Batch attribute configuration key (PdsBatchAttrib) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Batch attribute** configuration key controls access to the forms, fields, and functions that are used to work with batch attributes in process manufacturing logistics.

You can enable or disable this key in the **License configuration** form.

## Forms enabled by the configuration key

The following forms are available when the configuration key is enabled.
