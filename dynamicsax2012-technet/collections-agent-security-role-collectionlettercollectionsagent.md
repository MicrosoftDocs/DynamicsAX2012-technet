# Collections agent security role (CollectionLetterCollectionsAgent) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Collections agent security role represents a user who documents collections events and responds to collections inquiries.
