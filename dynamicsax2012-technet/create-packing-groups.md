# Create packing groups [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Inventory management** \> **Setup** \> **Packing material** \> **Packing groups**.

2.  Press CTRL+N to create a new packing group.

3.  In the **Packing group** field, type a short name for the group.

4.  In the **Name** field, type a description of the group.

5.  Close the form to save your changes.
