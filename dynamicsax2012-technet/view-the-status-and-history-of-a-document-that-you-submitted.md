# View the status and history of a document that you submitted 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Follow these steps to view the status and history of a document that you submitted to the workflow system for processing.

1.  Open the document.

2.  Review the message bar for information about the workflow process.

3.  Click **Actions** \> **View history** to open the **Workflow history** form, which displays the workflow status and history of the document. For more information about the **Workflow history** form, see [Workflow history (form)](https://technet.microsoft.com/en-us/library/hh597256\(v=ax.60\)).
