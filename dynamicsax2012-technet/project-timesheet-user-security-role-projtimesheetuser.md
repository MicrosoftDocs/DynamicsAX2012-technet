# Project timesheet user security role (ProjTimesheetUser) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Project timesheet user security role represents a user who enables creation and approval of project timesheets.
