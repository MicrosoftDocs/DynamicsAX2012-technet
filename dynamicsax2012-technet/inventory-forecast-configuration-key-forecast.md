# Inventory forecast configuration key (Forecast) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Inventory forecast** configuration key controls access to forms and functions that are used in forecasting.

If this configuration key is enabled, you can create forecasts for inventory, customers, and vendors.

You can enable or disable this key in the **License configuration** form.

## Forms enabled by the configuration key

The following forms are available when the configuration key is enabled.
