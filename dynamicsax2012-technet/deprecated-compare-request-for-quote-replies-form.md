# Deprecated: Compare request for quote replies form 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

In Microsoft Dynamics AX 2009, the **Compare request for quote replies** form (PurchRFQCaseCompare) lists all requests for quotations (RFQs). The user can select and compare multiple RFQ replies.
