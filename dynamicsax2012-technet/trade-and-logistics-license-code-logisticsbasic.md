# Trade and logistics license code (LogisticsBasic) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Trade and Logistics** license code enables access to functionality within trade and logistics.

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:

[Customer self-service configuration key (CSS)](customer-self-service-configuration-key-css.md)

[Sites purchasing configuration key (SitesPurchasing)](sites-purchasing-configuration-key-sitespurchasing.md)

[Trade configuration key (LogisticsBasic)](trade-configuration-key-logisticsbasic.md)

[Send electronically configuration key (AxdSend)](send-electronically-configuration-key-axdsend.md)


> [!NOTE]
> <P>The configuration keys listed here may have additional child configuration keys. Click the link for more information about a configuration key.</P>




> [!NOTE]
> <P>The Trade configuration key (LogisticsBasic) enables the Warehouse and Transportation configuration key (WHSandTMS).</P>




> [!NOTE]
> <P>Microsoft Dynamics AX 2012 R3 includes new Warehouse and Transportation management modules that are enabled by a single configuration key, along with the previously released Warehouse management II module. Users should enable the configuration key for <STRONG>ONLY ONE</STRONG> of these modules, but not both, in a single-instance, single-partition deployment. Although technically feasible, enabling the configuration keys for both of these modules in a single-instance, single-partition deployment is not supported by Microsoft.&nbsp;</P>
