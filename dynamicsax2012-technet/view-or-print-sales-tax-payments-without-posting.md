# View or print sales tax payments without posting 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Follow these steps to view or print sales tax payments without posting them.

1.  Click **General ledger** \> **Periodic** \> **Sales tax payments** \> **Sales tax payments**.

2.  Enter the criteria to use for the sales tax calculation and clear the **Update** check box.

3.  Click **OK** to run the calculation.

4.  In the **U.S. sales tax report** form, click **Destinations ...** to select how the report is delivered, and then click **OK**.
