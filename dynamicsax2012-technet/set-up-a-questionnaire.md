# Set up a questionnaire 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_


> [!NOTE]
> <P>Before you can set up a questionnaire, you must set up questions, answers, and questionnaire prerequisites. For more information, see <A href="designing-questionnaires.md">Designing questionnaires</A> and <A href="using-questionnaires.md">Using questionnaires</A>.</P>



1.  Click **Home** \> **Common** \> **Questionnaires** \> **Design** \> **Questionnaires**.

2.  Click **New** to create a new line.

3.  In the **Questionnaire** field, enter a unique identifier for the questionnaire, and in the **Description** field, enter a brief description of the questionnaire.

4.  In the **Type of questionnaire**, select the type of questionnaire to set up.
    
    For more information, see [About questionnaire types](about-questionnaire-types.md).

5.  Enter additional information, as needed.
    
    For more information, see [Questionnaires (form)](https://technet.microsoft.com/en-us/library/aa576998\(v=ax.60\)).
