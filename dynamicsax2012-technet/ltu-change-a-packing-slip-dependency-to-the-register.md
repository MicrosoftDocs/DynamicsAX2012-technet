# (LTU) Change a packing slip dependency to the register [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets** \> **Inquiries** \> **Packing slips**.

2.  Select the packing slip for which to change the dependency to the register, and then **Change dependency to register** to open the form.

3.  Click **OK** for the question **Change dependency to registry**. The dependency is automatically reversed.

4.  Click **OK** to apply the status change and return to the **Invoice journal** form.

5.  Press CTRL+S or close the form.
