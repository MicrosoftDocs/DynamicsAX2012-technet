# BUS Runtime configuration key (RuntimeBUS) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **ISV runtime** configuration key controls access to the application objects in add-on solutions that are provided to customers in the BUS layer.

If this configuration key is enabled, you can execute additional functionality in the BUS layer. For example, you can import Microsoft Dynamics AX models into the ISV layer when this key is enabled.

The configuration key is automatically enabled when the license key is entered into the system, and the key is not visible in the **License configuration** form.

You cannot enable or disable this key in the **License configuration** form.
