# Add a formula line to a batch order 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to make changes or add new items to formulas.

1.  Click **Production control** \> **Common** \> **Production orders** \> **All production orders**.

2.  Double-click the batch order to add a formula line to.

3.  On the **Action Pane**, on the **Production order** tab, in the **Order details** group, click **Formula** to open the **Formula lines-batch** form.

4.  Press CTRL+N to add a new line.

5.  In the **Item number** field, select the item to add to the formula.

6.  Optionally, enter the remaining information.
