# Customer anonymous security role (AnonymousCustomer) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Customer anonymous role represents an external user who browses products published in the sales catalog on the customer portal.
