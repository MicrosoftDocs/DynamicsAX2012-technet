# (RUS) Set up the transport tax code in the fixed assets record card 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

You can specify the selection criteria for the transport tax in the **Assets** form.


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets (Russia)** \> **Periodic** \> **Taxes filling in fixed asset table**.

2.  In the **Transport tax** field, select the sales tax code for transport tax.

3.  Click **Select** to open the **Assets** form to specify the selection criteria for fixed assets. You can select the asset group of the transport tax and the asset number. You can also specify the procurement date from which the asset will be assessed for taxation.

4.  Click **OK** to save the selection criteria in the **Assets** form.

5.  Click **OK** to close the form.
