# (RUS) Set up groups for posting fixed assets taxes 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets (Russia)** \> **Setup** \> **Tax reporting** \> **Group of posting of taxes**.

2.  Press CTRL+N to create a new line.

3.  In the **Ledger posting group** field, select a group for posting taxes into the ledger accounts.

4.  In the **Account for asset tax** field, select an account for the accrued tax on fixed assets.
    

    > [!NOTE]
    > <P>The <STRONG>Sales tax payable</STRONG> field displays the ledger account number that is specified for the selected ledger posting group.</P>



5.  Press CTRL+S or close the form.
