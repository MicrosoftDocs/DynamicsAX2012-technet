# Create a customer rebate group [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to create a customer rebate group. You can apply a customer rebate group to customers that qualify for each rebate in the group. By applying rebates to customers at the group level, you avoid having to assign each rebate individually.

1.  Click **Sales and marketing** \> **Setup** \> **Rebate program** \> **Customer** \> **Customer rebate groups**.

2.  Press CTRL+N to create a new customer rebate group.

3.  In the **Customer rebate group** field, enter a unique name or identifier.

4.  In the **Description** field, enter a description for the customer rebate group.
