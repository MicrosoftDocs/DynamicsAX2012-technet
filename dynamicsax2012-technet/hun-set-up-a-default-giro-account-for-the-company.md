# (HUN) Set up a default GIRO account for the company 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Organization administration** \> **Setup** \> **Organization** \> **Legal entities**.

2.  Click the **Bank account information** FastTab.

3.  Under the **Bank accounts** field group, in the **Giro account** field, enter the primary GIRO account of the company.

4.  Press CTRL+S or close the form.
