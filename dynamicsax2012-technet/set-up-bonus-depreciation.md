# Set up bonus depreciation 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

You can use the **Special depreciation allowance** form to set up bonus depreciation. Bonus depreciation records are used to calculate bonus depreciation amounts through the bonus depreciation proposal process.

Bonus depreciation is available only for depreciation books. You do not have to set up bonus depreciation records. You can instead enter manual bonus depreciation transactions.

1.  Click **Fixed assets** \> **Setup** \> **Depreciation** \> **Special depreciation allowance**.

2.  Click **New** and enter a unique identifier in the **Bonus** field.

3.  Enter a description.

4.  Enter information in either the **Percentage** or the **Amount** field.
