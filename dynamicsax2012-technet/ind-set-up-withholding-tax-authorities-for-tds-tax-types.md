# (IND) Set up withholding tax authorities for TDS tax types 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **General ledger** \> **Setup** \> **Withholding tax** \> **India** \> **Withholding tax authorities**.

2.  Select **TDS** in the **Tax type** field to set up withholding tax authorities for TDS tax type.

3.  Enter a name for the TDS authority in the **Withholding tax authority** field.

4.  Enter a description for the TDS authority in the **Description** field.

5.  Select the TDS authority vendor account in the **Vendor account** field.
    

    > [!NOTE]
    > <P>The bank name to deposit the TDS amount that is payable to the TDS authority vendor must be defined in the <STRONG>Vendor bank accounts</STRONG> form.</P>
