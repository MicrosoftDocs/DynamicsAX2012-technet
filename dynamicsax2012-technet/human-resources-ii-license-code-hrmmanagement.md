# Human resources II license code (HRMManagement) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Human resource II** license code enables access to some of the functionality in the following Human resources areas:

  - Absence

  - Recruiting
