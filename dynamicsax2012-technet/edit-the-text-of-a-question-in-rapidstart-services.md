# Edit the text of a question in RapidStart Services 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Beginning October 1, 2015, RapidStart services will no longer be supported. For more information, see the announcement on CustomerSourcehttps://mbs.microsoft.com/partnersource/northamerica/news-events/news/RapidStartServicesDiscontinued or PartnerSourcehttps://mbs.microsoft.com/customersource/northamerica/news-events/news-events/news/RapidStartServicesDiscontinued.

To modify a question in Microsoft Dynamics ERP RapidStart Services, use the same form that you created the question in.


> [!NOTE]
> <P>You cannot modify questions that belong to a published functional area. For information about how to unpublish a functional area, see <A href="unpublish-a-functional-area-in-rapidstart-services.md">Unpublish a functional area in RapidStart Services</A>.</P>



1.  In the RapidStart Services header, click **Design**, and then click **Functional area**.

2.  In the **Functional area** list, select the functional area that contains the question that you want to modify.

3.  In the **Question group name** column, click the name of the question group that contains the question that you want to modify.

4.  In the **Question group** list, select the name of the question that you want to modify.

5.  Change the question as required.
    

    > [!TIP]
    > <P>For information about how to configure questions, see <A href="create-questions-and-question-actions-in-rapidstart-services.md">Create questions and question actions in RapidStart Services</A>.</P>



6.  Click **Save**.
