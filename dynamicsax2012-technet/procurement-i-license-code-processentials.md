# Procurement I license code (ProcEssentials) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Procurement I** license code enables access to some procurement and sourcing features of Microsoft Dynamics AX.
