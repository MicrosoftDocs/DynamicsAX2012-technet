# (RUS) Set up the assessed tax code in the fixed assets record card 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

Under Section 30 of the RF tax code on Company Property, movable and real property that is listed in the company account books as fixed assets is considered as assets of taxation.


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets (Russia)** \> **Periodic** \> **Taxes filling in fixed asset table**. click **assessed tax and transport tax filling.**

2.  In the **Assessed tax** field, select the sales tax code for assessed tax.

3.  Click **Select** to open the **Assets** form to specify the selection criteria for the assessed tax. You can select the asset group of the assessed tax and the asset number. You can also specify the procurement date from which the asset will be assessed for taxation.

4.  Click **OK** to save the selection criteria in the **Assets** form.

5.  Click **OK** to close the form.
    

    > [!NOTE]
    > <P>The <STRONG>Fixed Assets Code</STRONG> field displays the value that is specified in the <STRONG>asset tax and transport tax filling in fixed asset table</STRONG> form in the <STRONG>Fixed assets record card</STRONG>.</P>
