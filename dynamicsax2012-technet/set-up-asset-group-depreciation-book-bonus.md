# Set up asset group depreciation book bonus [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use the **Fixed asset group/depreciation book special depreciation allowance** form to set up an asset group depreciation book bonus.

1.  Click **Fixed assets** \> **Setup** \> **Fixed asset groups**.

2.  Select a fixed asset group.

3.  Click **Depreciation books**.

4.  Click **Special depreciation allowance**.

5.  Enter a priority number. If there is more than one bonus depreciation for an asset, this determines the order in which the bonuses are taken.

6.  Change the percentage or amount, if you have to.
