# Retail headquarters configuration key (RetailHeadquarters) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack_

The **Retail Headquarters** configuration key controls access to **Retail Headquarters** forms and functions.

You can enable or disable this key in the **License configuration** form.


> [!NOTE]
> <P>To use the <STRONG>Retail</STRONG> module, all retail configuration keys must be enabled except the configuration keys in the <STRONG>Call center</STRONG> group. The <STRONG>Call center</STRONG> configuration keys are required only if you intend to use the <STRONG>Call center</STRONG> channel type.</P>



## Forms enabled by the configuration key

The following forms are available when the configuration key is enabled.
