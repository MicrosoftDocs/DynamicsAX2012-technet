# Find records 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use one of the following methods to find data in Microsoft Dynamics AX:

  - Filter records in a grid. For more information, see [Filter records](filter-records.md).

  - Sort records in a grid. For more information, see [Sort records](sort-records.md).

  - Use Enterprise Search. For more information, see [Using search](using-search.md)
