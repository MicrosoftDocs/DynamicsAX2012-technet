# Keep update objects 6.2 (SysDeletedObjects62) 


Some objects in Microsoft Dynamics AX 2012 might be removed during upgrade to Microsoft Dynamics AX 2012 R2. These objects are given a prefix of DEL\_ and are assigned to the **Keep update objects 6.2** configuration key. When you clear the check box for this configuration key in the **License configuration** form, the objects assigned to this configuration key are deleted.


> [!WARNING]
> <P>Never manually delete tables and fields from your database.</P>



You can enable or disable this key in the **License configuration** form.
