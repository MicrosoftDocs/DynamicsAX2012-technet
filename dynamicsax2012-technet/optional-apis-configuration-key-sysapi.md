# Optional APIs configuration key (SysAPI) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Optional APIs** configuration key controls access to the **Dynamic data exchange api** and **Pipe client api** system classes.

Access to the rest of the system classes is controlled by the **X++ development** configuration key.

You cannot enable or disable this key in the **License configuration** form.
