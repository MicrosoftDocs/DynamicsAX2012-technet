# Create document types for applicant communication 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Before you can create document templates and application email templates, you must create the document types that will be used by each. For instructions to create a new document type, see [Configure document management](configure-document-management.md).

When you create the new document type for an applicant email template or document template, be sure to select the options listed in the following table.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>Field</p></th>
<th><p>Selection</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p><strong>Class description</strong></p></td>
<td><ul>
<li><p><strong>Create application document</strong> – Select this option for document templates.</p></li>
<li><p><strong>Simple note</strong> – Select this option for email templates.</p></li>
</ul></td>
</tr>
<tr class="even">
<td><p><strong>Group</strong></p></td>
<td><ul>
<li><p><strong>Document</strong> – Select this option for document templates.</p></li>
<li><p><strong>Note</strong> – Select this option for email templates.</p></li>
</ul></td>
</tr>
</tbody>
</table>


After you create the document types to use to communicate with applicants, you must select those document types in the **Recruitment** area on the **Human resources parameters** form.
