# Microsoft Office Outlook Synchronization configuration key (smmOutlook) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Microsoft Outlook synchronization** configuration key allows you to synchronize your contacts, calendar appointments, and tasks between Microsoft Outlook and Microsoft Dynamics AX.
