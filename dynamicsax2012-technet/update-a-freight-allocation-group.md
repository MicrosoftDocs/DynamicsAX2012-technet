# Update a freight allocation group 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to update information for a freight allocation group.

1.  Click **Sales and marketing** \> **Setup** \> **Rebate program** \> **Item** \> **Item freight groups**.

2.  Select the freight allocation group that you want to update.

3.  In the **Description** field, modify the description for the freight allocation group.
    

    > [!NOTE]
    > <P>If you want to change the value in the <STRONG>Freight allocation group</STRONG> field, you must delete the record and create a new one.</P>
