_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The workflow system uses batch jobs to process messages, determine due dates for work items, and process notifications for line items. Complete the following procedures to create a batch group and configure the batch jobs.

[Create a batch group for the workflow system](create-a-batch-group-for-the-workflow-system.md)
