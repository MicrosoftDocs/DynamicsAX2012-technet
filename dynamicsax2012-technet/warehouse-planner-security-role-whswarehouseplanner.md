# Warehouse planner security role (WHSWarehousePlanner) 


_**Applies To:** Microsoft Dynamics AX 2012 R3_

The Warehouse planner security role represents a user who does warehouse planning.


> [!NOTE]
> <P>This role is available only if Microsoft Dynamics AX 2012 R3 is installed.</P>
