# (HUN) Set up charges codes for Intrastat [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Accounts receivable** \> **Setup** \> **Charges** \> **Charges code**.

2.  Create a charges code.

3.  Select the **Include into Intrastat** check box to include the charges posted for charges codes in the Intrastat report.

4.  Press CTRL+S or close the form.
