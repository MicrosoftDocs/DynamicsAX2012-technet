# Project III configuration key (Project3) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Project III** configuration key controls access to forms and functions that are used to complete the following tasks in **Project management and accounting**:

  - Request an advance payment from a customer.

  - Create and maintain amounts retained by customers.

  - Create and maintain amounts withheld from payments to a vendor until you are paid by a customer.

  - Set up indirect costs for a project.

  - Set up billing rules to calculate and schedule customer invoices for projects.

  - Integrate and synchronize project quotations and project information between Microsoft Dynamics AX and Microsoft Dynamics CRM.
    
    This task is available only if you have installed Microsoft Dynamics AX 2012 R3 or cumulative update 6 or later for AX 2012 R2.

You cannot enable or disable this key in the **License configuration** form.
