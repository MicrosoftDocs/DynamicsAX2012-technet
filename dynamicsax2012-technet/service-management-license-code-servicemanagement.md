# Service management license code (ServiceManagement) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Service management** license code enables access to the **Service management** module in Microsoft Dynamics AX.
