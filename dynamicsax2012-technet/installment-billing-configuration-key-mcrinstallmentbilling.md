# Installment billing configuration key (MCRInstallmentBilling) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3_

The **Installment billing** configuration key controls access to installment billing functionality in the **Call center** module.

You can enable or disable this key in the **License configuration** form.

## Forms enabled by the configuration key

The following forms are available when the configuration key is enabled.
