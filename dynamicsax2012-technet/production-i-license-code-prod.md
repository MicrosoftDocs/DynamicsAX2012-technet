# Production I license code (Prod) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Production I** license code enables access to some of the production features of Microsoft Dynamics AX. These features include the basic functionality for creating and controlling production orders.
