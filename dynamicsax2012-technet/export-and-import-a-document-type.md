# Export and import a document type 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

## Export document types

1.  Click **Organization administration** \> **Setup** \> **Document management** \> **Document types**.

2.  Click **Import/Export** and select **Export**.

3.  In **File name**, type the path and the name of the export file, or select a folder on your computer or network, and then click **OK**.

## Import document types

1.  Click **Organization administration** \> **Setup** \> **Document management** \> **Document types**.

2.  Click **Import/Export** and select **Import**.

3.  In **File name**, type the path and the name of the import file, or select a file on your computer or network.

4.  When prompted to delete tables before import, click **Yes to all**, and then click **OK**.

5.  Click **Yes** to confirm.
