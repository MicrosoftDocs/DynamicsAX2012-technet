# Personnel actions configuration key (HcmPersonnelActions) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

The **Personnel actions** configuration key allows you to use an actions process to gather details and track when your users:

  - Create or modify positions.

  - Hire employees, transfer employees, change employee compensation, or terminate an employee.

You can also require that these additions and changes go through a review process.


> [!TIP]
> <P>You can enable or disable this key in the <STRONG>License configuration</STRONG> form.</P>
