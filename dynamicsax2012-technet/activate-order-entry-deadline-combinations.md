# Activate order entry deadline combinations 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to activate combinations of order entry deadline groups and sites.

1.  Click **Inventory management** \> **Setup** \> **Distribution** \> **Activate order entry deadline combinations**.

2.  Select the check boxes that correspond to the combinations to activate.

<colgroup>
<col style="width: 33%" />
<col style="width: 33%" />
<col style="width: 33%" />
</colgroup>
<tbody>
<tr class="odd">
<td></td>
<td><p><strong>Order group</strong></p></td>
<td><p><strong>All order groups</strong></p></td>
</tr>
<tr class="even">
<td><p><strong>Site</strong></p></td>
<td><p>Select this check box to apply order entry deadlines to a specific site and a specific order entry deadline group.</p></td>
<td><p>Select this check box to apply order entry deadlines to a specific site and all order groups.</p></td>
</tr>
<tr class="odd">
<td><p><strong>All sites</strong></p></td>
<td><p>Select this check box to apply order entry deadlines to all sites and a specific order entry deadline group.</p></td>
<td><p>This check box for all sites and all order entry deadline groups is always selected and cannot be cleared.</p></td>
</tr>
</tbody>
</table>


The selected combinations will be available in the **Order entry deadlines** form.
