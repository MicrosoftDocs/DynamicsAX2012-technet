# (HUN) Set up import methods for account statements 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **General ledger** \> **Setup** \> **Journals** \> **Methods of importing account statements**.
    

    > [!NOTE]
    > <P>For more information, see "Import methods for account statements (form)" in the Applications and Business Processes Help.</P>



2.  In the **Import format** field, select **M\_cash (HU)**.

3.  Press CTRL+S or close the form.
