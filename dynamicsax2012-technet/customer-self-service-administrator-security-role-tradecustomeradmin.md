# Customer self-service administrator security role (TradeCustomerAdmin) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Customer self-service administrator security role represents an external user who maintains external party access and information.

## Duties
