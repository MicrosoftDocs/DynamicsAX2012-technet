# Centralized customer statements configuration key (CustCentralizedCustStatement) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Centralized customer statements** configuration key controls the ability to create customer statements on behalf of multiple legal entities in your organization. If this configuration key is enabled, you can use the **Customer account statement** report to print centralized statements.

You can enable or disable this key in the **License configuration** form.
