# (CZE) Acquisition of a new fixed asset 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Fixed assets** \> **Journals** \> **Fixed assets**.

2.  Enter the required details and post the journal.


> [!NOTE]
> <P>This step changes the status of the fixed asset to <STRONG>Open</STRONG> in the <STRONG>Status</STRONG> field in the <STRONG>Value models</STRONG>. The asset is now eligible for depreciation. You can create the depreciation proposal using the <STRONG>Depreciation proposal</STRONG> form.</P>
