# Install-AXReportInstanceExtensions

## Install-AXReportInstanceExtensions

If multiple instances of Microsoft SQL Server Reporting Services are running on the same computer, automates the process of integrating those instances with Microsoft Dynamics AX.

## Syntax

    Parameter Set: Default
    Install-AXReportInstanceExtensions [-Credential <PSCredential> ] [-ReportServerInstanceName <String> ] [-ServicesAOSName <String> ] [-ServicesAOSWSDLPort <Int32> ] [-ServicesFilePath <String> ] [ <CommonParameters>]

  
  

## Detailed description

If multiple instances of Microsoft SQL Server Reporting Services are running on the same computer, the Install-AXReportInstanceExtensions cmdlet automates the process of integrating those instances with Microsoft Dynamics AX. The cmdlet installs Reporting Services extensions on secondary Reporting Services instances.
