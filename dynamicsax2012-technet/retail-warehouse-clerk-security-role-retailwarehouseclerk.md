# Retail warehouse clerk security role (RetailWarehouseClerk) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack_

The Retail warehouse clerk security role represents a user who is responsible for recording quantities in Enterprise Portal for picking, receiving, and counting documents.
