# Retail Commerce Data Exchange backward compatibility configuration key (RetailCDXBackwardCompatibility) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

The **Retail Commerce Data Exchange backward compatibility** configuration key enables you to use previous versions of Retail POS to exchange data with AX 2012 R3 or AX 2012 R3. This functionality is typically used during an upgrade.

You can enable or disable this key in the **License configuration** form.

## Forms enabled by the configuration key

The following forms are available when the configuration key is enabled.
