# Create service agreement groups 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.

2.  Press CTRL+N to create a new service agreement group.

3.  In the **Service agreement group** field, enter an ID for the service agreement group.

4.  In the **Description** field, enter a description for the service agreement group.

5.  Close the form to save your changes.


> [!NOTE]
> <P>If a service agreement group is defined in the <STRONG>Service agreement group</STRONG> field of the <STRONG>Service management parameters</STRONG> form, the program suggests this group for all new service agreements that you create.</P>
