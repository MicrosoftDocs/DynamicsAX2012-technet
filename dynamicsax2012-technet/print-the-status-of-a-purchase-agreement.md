# Print the status of a purchase agreement 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Procurement and sourcing** \> **Reports** \> **Status** \> **Purchase agreement fulfillment**.

2.  Print the report using one of the following methods:
    
      - Click **OK** to print a report that displays the status of all purchase agreements.
    
      - Click **Select** to define criteria for the report. For example, you can print the status of a specific purchase agreement. Then click **OK** to print the report.
