# Start pallet transports 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

This topic applies to features in the **Inventory management** module. It does not apply to features in the [Warehouse management](warehouse-management.md) module.

1.  Click **Inventory management** \> **Common** \> **Pallet transports**. Select the pallet transport that you want to start.

2.  Click **Start transport**.

3.  Specify the appropriate options, and then click **OK**.
