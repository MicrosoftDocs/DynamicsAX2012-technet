# Data import/export framework administrator security role (DMFAdministrator) [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3_

The Data import/export framework administrator security role represents a user who administers the Microsoft Dynamics AX Data Import/Export Framework.


> [!NOTE]
> <P>This role is available only if Microsoft Dynamics AX 2012 R3 is installed.</P>
