# (BEL) Set up State 11 information 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

To print the State 11 report, you must set up State 11 information.

1.  Click **Cash and bank management** \> **Setup** \> **State 11**.

2.  Press CTRL+N to create a bank account for State 11 reporting.

3.  In the **Bank account** field, select the bank account number to use for State 11 reporting.

4.  Select the **State 11** check box to enable reporting for the selected bank account.

5.  In the **Sequence number** field, you can enter a sequence code, if required.

6.  Close the form to save your changes.
