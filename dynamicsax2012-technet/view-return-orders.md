# View return orders 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

You can use the **All return orders** and **Return order** pages to view information about return orders. You create these when customers return items that they have purchased.

1.  Click **Sales** on the top link bar, and then click **Return orders** on the Quick Launch. On the **All return orders** list page, you can view a list of return orders.

2.  Select an order. On the **Action Pane**, on the **Return order** tab, in the **Maintain** group, click **View**, or click a return order link. The **Return order** page is displayed, where you can view the details of the return order. This includes the line items.
