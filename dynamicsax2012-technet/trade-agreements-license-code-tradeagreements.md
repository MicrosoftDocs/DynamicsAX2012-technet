# Trade agreements license code (TradeAgreements) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Trade agreements** license code controls access to functionality that relates to supplementary item and price and discount agreements.
