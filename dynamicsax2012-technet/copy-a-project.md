# Copy a project 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use the **Project Copy Wizard** to copy a project or a project tree hierarchy.

## Make a project available to be copied

1.  Click **Project management** on the top link bar, and then in the **Project name** column, click the project that you want to work with.

2.  On the **Action Pane**, in the **Maintain** group, click **Edit**.

3.  On the **Edit project** page, on the **General** FastTab, under **Project hierarchy**, select the **Project template** check box.

4.  Click **Save and close**.


> [!NOTE]
> <P>To make a project tree hierarchy available for copying, only the parent project must be defined as a project template.</P>



## Copy a project

1.  Click **Project management** on the top link bar, and then in the **Project name** column, click the project that you want to work with.

2.  On the **Action Pane**, on the **Project** tab, in the **New** group, click **Copy project**.

3.  Follow the instructions in the **Project Copy Wizard**.
