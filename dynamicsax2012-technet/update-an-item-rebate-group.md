# Update an item rebate group 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use this procedure to update information for an item rebate group.

1.  Click **Sales and marketing** \> **Setup** \> **Rebate program** \> **Item** \> **Item rebate groups**.

2.  Select the item rebate group that you want to update.

3.  In the **Description** field, modify the description for the item rebate group.
    

    > [!NOTE]
    > <P>If you want to change the value in the <STRONG>Item rebate group</STRONG> field, you must delete the record and create a new one.</P>
