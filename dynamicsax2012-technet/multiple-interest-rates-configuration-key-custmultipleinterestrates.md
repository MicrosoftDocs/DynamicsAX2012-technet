# Multiple interest rates configuration key (CustMultipleInterestRates) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Multiple interest rates** configuration key is obsolete and is used only for backward compatibility. Use the **Interest** configuration key, instead.

You can enable or disable this key in the **License configuration** form.
