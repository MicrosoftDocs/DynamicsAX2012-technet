# (BEL) Create and transfer transactions to the BLWI 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use the **BLWI transaction** form to view and update BLWI transactions before you transfer data to the BLWI report. Use the **BLWI** form to transfer the transactions to the **BLWI transaction** form.

1.  Click **Organization administration** \> **Periodic** \> **Foreign trade** \> **BLWI**.

2.  Click **Transfer** to open the **BLWI** form.

3.  In the **Survey code** field, select a survey code.

4.  In the **Date** field, select a transaction date.

5.  Select the **Delete** check box to delete the transactions that are already transferred to the **BLWI transaction** form.

6.  Click **OK** to transfer the transactions to the **BLWI transaction** form.

7.  Close the form to save your changes.
