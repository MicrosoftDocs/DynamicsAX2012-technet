# View the work items in the queues that you administer 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

To view the work items in the queues that you administer, follow these steps.

1.  Click **Home** \> **Common** \> **Work items** \> **Work items assigned to queues I administer**.

2.  View the work items in the list.
