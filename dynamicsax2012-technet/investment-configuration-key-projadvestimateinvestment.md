# Investment configuration key (ProjAdvEstimateInvestment) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Investment** configuration key controls access to Investment project functions.


> [!NOTE]
> <P>To enable this configuration key, you must first have the <STRONG>Expense management</STRONG> configuration key enabled.</P>



You can enable or disable this key in the **License configuration** form.
