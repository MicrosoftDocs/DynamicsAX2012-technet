# Create service-agreement lines manually 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**. Select a service agreement, and then click the **Edit** button on the Action Pane

2.  From the **Service agreements** form, add a service-agreement line in the **Lines** grid.

3.  Enter the appropriate information for the service-agreement line. For more information, see [Service agreements (form)](https://technet.microsoft.com/en-us/library/aa617823\(v=ax.60\)).

4.  Press CTRL+S to save the line, and then close the form.
