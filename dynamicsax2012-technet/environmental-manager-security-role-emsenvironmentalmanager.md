# Environmental manager security role (EMSEnvironmentalManager) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Environmental manager security role represents a user who enables and reviews the environmental process.
