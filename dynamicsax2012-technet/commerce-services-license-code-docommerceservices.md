# Commerce Services license code (DOCommerceServices) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Commerce Services** license code enables access to Commerce Services for Microsoft Dynamics ERP.
