# Array default values [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

A modeling variable group of the **Array** type is displayed as a table in which each variable in the group is shown as one column. You can set up default values for the array elements.


> [!NOTE]
> <P>This information applies only to Product builder.</P>



## Default values

For modeling variables that belong to an array variable group, you can click **Default value** in the **Product model** form to assign default values to each array element.
