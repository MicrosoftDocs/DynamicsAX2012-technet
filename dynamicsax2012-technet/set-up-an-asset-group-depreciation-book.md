# Set up an asset group depreciation book 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

You can use the **Fixed asset group/depreciation book** form to associate depreciation books to fixed asset groups.

1.  Click **Fixed assets** \> **Setup** \> **Fixed asset groups**.

2.  Click **Depreciation books**.

3.  In the **Depreciation book** field, select a depreciation book that you want to associate with the fixed asset group.
