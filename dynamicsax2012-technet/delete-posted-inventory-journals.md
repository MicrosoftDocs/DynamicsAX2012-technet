# Delete posted inventory journals 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Inventory management** \> **Periodic** \> **Clean up** \> **Inventory journals cleanup**.

2.  Select the types of inventory journals to delete.

3.  Click the **Batch** tab if you want to define parameters to delete old inventory journals regularly.

4.  Click **OK** to run the deletion process.
