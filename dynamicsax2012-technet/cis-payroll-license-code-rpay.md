# CIS Payroll license code (RPay) 


_**Applies To:** Microsoft Dynamics AX 2012 R3_

The **CIS Payroll** license code and all configuration keys for the license code have been deprecated in Microsoft Dynamics AX 2012.

The **CIS Payroll** license code enables access to some of the following CIS payroll functionality:

  - Business trip calculation

  - Contract calculation

  - Employee average earning calculation

  - Lucre

  - Moonlighters correction

  - Periodical payments

  - Sick lists

  - Standard deductions

  - Tax calculations

  - Timesheet

  - Vacation payment calculation

## Configuration keys enabled by this license code

The following configuration keys are controlled by this license code:
