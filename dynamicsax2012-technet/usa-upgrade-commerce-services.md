# (USA) Upgrade Commerce Services 


_**Applies To:** Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The topics in this section provide information about how to upgrade an account and data for the account in Commerce Services for Microsoft Dynamics ERP.

If you had a Commerce Services account in Microsoft Dynamics AX 2012, and you upgrade to Microsoft Dynamics AX 2012 Feature Pack, you must upgrade the Commerce Services account and its data.

There are three steps in this upgrade process:

1.  [(USA) Activate Commerce Services](usa-activate-commerce-services.md)

2.  [(USA) Migrate the Commerce Services account](usa-migrate-the-commerce-services-account.md)

3.  [(USA) Migrate the Commerce Services data](usa-migrate-the-commerce-services-data.md)

For information about how to work with Commerce Services, see [(USA) Sell products online with Commerce Services](usa-sell-products-online-with-commerce-services.md).


> [!NOTE]
> <P>This feature is not available if Microsoft Dynamics AX 2012 R3 is installed.</P>
