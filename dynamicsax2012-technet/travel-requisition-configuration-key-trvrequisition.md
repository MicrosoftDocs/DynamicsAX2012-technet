# Travel requisition configuration key (TrvRequisition) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Travel requisition** configuration key allows workers to create, and submit for approval, travel requisitions for future expenses that will be incurred and entered on an expense report.

Policies regarding the submission of travel requisitions can be created to enforce the expense policies set by your organization.

You can enable or disable this key in the **License configuration** form.

The following forms are available when the configuration key is enabled.
