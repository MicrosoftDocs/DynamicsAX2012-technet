# Currency configuration key (Currency) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Currency** configuration key controls access to all currency forms and functions.

This parent configuration key cannot be disabled. However, you can disable the child configuration keys to disable individual features.

You cannot enable or disable this key in the **License configuration** form.
