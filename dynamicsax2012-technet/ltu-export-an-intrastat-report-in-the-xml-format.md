# (LTU) Export an Intrastat report in the XML format 


[!INCLUDE[archive-banner](includes/archive-banner.md)]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Organization administration** \> **Periodic** \> **Foreign trade** \> **Intrastat**. Click **Output** and then click **Report** to open the **Intrastat report** form.

2.  In the **Directory** field, select the location to save the file.

3.  In the **Direction** field, select the direction type from these options:
    
      - **Arrivals** – Purchase, sales returned, or acquisition transactions.
    
      - **Dispatches** – Sales, purchases returned, or disposal transactions.

4.  In the **Version** field, enter the report version. If you are creating the report for the first time, enter the value **0.** Otherwise, enter the number of the new version of the report.
    

    > [!NOTE]
    > <P>When you create a report for the first time, the numbers for the report lines are displayed in the <STRONG>Item sequence number</STRONG> field in the <STRONG>Intrastat</STRONG> form. If you create subsequent versions of the same report, the existing values in the <STRONG>Item sequence number</STRONG> field are used.</P>



5.  Enter the required details, and then click **OK** to generate the Intrastat report and close the **Intrastat report** form.
