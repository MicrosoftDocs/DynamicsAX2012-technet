# Enable license code authoring and editing license code (SysLicenseCodeEditing) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Enable license code authoring and editing** license code enables license code authoring and editing.

This license code enables an association between the ISV license code and the ISV certificate via the AOT. This association will enable the user to map the license code to the Authenticode certificate property.

When this license code is installed, developers can to map their license code to their LicenseCode CertificateName property.
