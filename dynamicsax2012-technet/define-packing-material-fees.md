# Define packing material fees [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Inventory management** \> **Setup** \> **Packing material** \> **Packing material fees**.

2.  Select a code in the **Code** field.

3.  Enter the period of validity for the fee by using the **From date** and **To date** fields.

4.  In the **Price** field, enter the fee to assign to the selected packing material code.
    

    > [!NOTE]
    > <P>(NLD) You can use up to five decimal places. This option is available only to legal entities whose primary address is in the Netherlands.</P>



5.  Select the currency in the **Currency** field.
