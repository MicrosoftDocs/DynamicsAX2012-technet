# Var Runtime configuration key (RuntimeVAR) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **VAR runtime** configuration key controls access to the application objects in add-on solutions that are provided in the VAR layer.

With the configuration key enabled, you can execute additional functionality in the BUS layer. For example, you can import Microsoft Dynamics AX models into the VAR layer when this key is enabled.

You cannot enable or disable this key in the **License configuration** form.
