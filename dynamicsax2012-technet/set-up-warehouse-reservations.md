# Set up warehouse reservations 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

1.  Click **Accounts receivable** \> **Common** \> **Customers** \> **All customers**.

2.  Select the customer for which you want to specify a default warehouse.

3.  On the **Sales order** FastTab, select the default warehouse in the **Warehouse** list.


> [!NOTE]
> <P>If you are using warehouse management, you typically reserve inventory from the warehouse that it is most convenient to deliver from.</P>
