# Machine operator security role (JmgMachineOperator) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The Machine operator security role represents a user who works on production orders and makes registrations in the Manufacturing execution module.
