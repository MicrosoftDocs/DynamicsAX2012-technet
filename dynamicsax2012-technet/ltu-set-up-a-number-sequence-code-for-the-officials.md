# (LTU) Set up a number sequence code for the officials 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

You must assign a number sequence code for the company officials.


> [!NOTE]
> <P>This topic has not been fully updated for Microsoft Dynamics AX 2012 R2.</P>



1.  Click **Organization administration** \> **Setup** \> **Organization** \> **Legal entities**.

2.  Click the **Number sequence** tab.

3.  In the **Number sequence code** field, select the number sequence code for the **Reference** type **Officials session ID**.

4.  Press CTRL+S or close the form.
