# Financials II license code (LedgerAdvanced) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Financials II** license code enables access to some of the following functionality in General ledger:

  - Allocations

  - Cash flow forecasts

  - Consolidations

  - Intercompany accounting
