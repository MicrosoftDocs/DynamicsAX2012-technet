# Questionnaire license code (KMQuestionnaire) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Questionnaire** license code enables access to the questionnaire functionality in Microsoft Dynamics AX.
