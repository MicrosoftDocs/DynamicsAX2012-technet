# Deprecated: (BRA) Fuel code [AX 2012]


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2_

For Brazilian installations, Microsoft Dynamics AX 2009 provides country-specific modifications that support the fuel code that is required for gas and oil distribution segments.
