# Human resources I license code (HRMAdministration) 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

The **Human resource I** license code enables access to some of the functionality in the following Human resources areas:

  - Americans with Disabilities Act (ADA)

  - Compensation

  - Payroll

  - I-9 Form

  - Benefits

  - Competencies

  - Injury or Illness

  - Worker information
