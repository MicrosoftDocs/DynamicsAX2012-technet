# Modify a workflow 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Follow these steps to modify a workflow.

1.  Open the module that contains the workflow to modify.

2.  Click **Setup** \> **\[Module name\] workflows**. A list page is displayed.

3.  Select the workflow to modify.

4.  On the Action Pane, click **Edit**. The workflow editor opens and displays the workflow that you selected.

5.  Modify the workflow. For more information about how to modify a workflow and the elements in it, see the following topics:
    
      - [Create a workflow](create-a-workflow.md)
    
      - [Configure the properties of a workflow](configure-the-properties-of-a-workflow.md)
    
      - [Configure workflow elements](configure-workflow-elements.md)
