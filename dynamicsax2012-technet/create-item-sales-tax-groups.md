# Create item sales tax groups 


_**Applies To:** Microsoft Dynamics AX 2012 R3, Microsoft Dynamics AX 2012 R2, Microsoft Dynamics AX 2012 Feature Pack, Microsoft Dynamics AX 2012_

Use the **Item sales tax groups** form to create an item sales tax group.

1.  Click **General ledger** \> **Setup** \> **Sales tax** \> **Item sales tax groups**.

2.  To create a sales tax group, click **New**.

3.  In the **Item sales tax group** field, enter a unique identifier for the item sales tax group.

4.  In the **Description** field, enter a name or description of the item sales tax group.

5.  In the **Reporting type** field, select a reporting type, which determines where the transaction line amount is included on the European Union (EU) sales list.

6.  Click the **Setup** FastTab. In the **Sales tax code** field, select a sales tax code.

7.  To add another sales tax code to the item sales tax group, click **Add**.
